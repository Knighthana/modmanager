# repo_logs — 禁止主动阅读

本目录存放历史工作日志，作为冷备份。

**Agent 约束**：除非收到明确指令（如"查看某日的日志"），否则**禁止**阅读此目录下的任何文件。日常工作中不得以任何理由主动检索、引用或依赖本目录内容。

## 触发阅读的明确指令示例
- "查看 2026-04-30 的日志"
- "列出 repo_logs 中所有文件"
- 其他显式要求读取本目录内容的指令

## 非触发阅读的情形
- "检查项目状态"
- "排查 XX 问题"
- "有什么没完成的任务"
- 任何未显式提及 `repo_logs` 或 "日志" 的指令

违反此约定将导致上下文污染。
